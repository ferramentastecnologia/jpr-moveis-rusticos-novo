## IMPORTANTE: API Figma

A Figma REST API **NÃO permite criar arquivos novos** via POST.

**Solução:** Criamos um **Figma Plugin Script** que executa no Figma Desktop e cria tudo automaticamente!

---

## QUICK START (3 minutos)

### Método 1: Plugin Script (MAIS RÁPIDO)

1. Abra o Figma Desktop
2. Crie novo arquivo: "Luis Alves Mesas - Landing Page"
3. Menu: Plugins > Development > New Plugin
4. Cole o código de: `figma-plugin-script.js`
5. Clique "Run"
6. PRONTO! Tudo criado automaticamente

### Método 2: Figma Tokens Plugin

1. Instale plugin "Figma Tokens"
2. Importe: `design-tokens.json`
3. Apply to document
4. Crie componentes manualmente

### Método 3: Manual

Siga instruções detalhadas em: `FIGMA-INSTRUCOES.md`

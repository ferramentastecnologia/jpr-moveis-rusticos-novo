## ARQUIVOS ESSENCIAIS (Use estes!)

### 1. IMPLEMENTAÇÃO NO FIGMA

**figma-plugin-script.js** (11KB)
- Script executável no Figma Plugin Dev Mode
- Cria TUDO automaticamente (Design System + Components + Layouts)
- Método mais rápido e recomendado

**Como usar:**
1. Abra Figma Desktop
2. Plugins > Development > New Plugin
3. Cole este código
4. Run
5. PRONTO!

### 2. DESIGN TOKENS

**design-tokens.css** (2.5KB)
- Variáveis CSS prontas para desenvolvimento
- 8 cores, 10 tipografias, sombras, espaçamentos
- Use diretamente em HTML/CSS/React

**design-tokens.json** (5.8KB)
- Formato W3C Design Tokens Standard
- Compatível com Figma Tokens plugin
- Compatível com Style Dictionary

## FLUXO DE TRABALHO RECOMENDADO

### Fase 1: Setup Rápido (5 minutos)

1. Leia: **README-FIGMA.md**
2. Abra no navegador: **figma-preview.html**
3. Execute no Figma: **figma-plugin-script.js**

### Fase 2: Customização (30 minutos)

1. Adicione fotos reais das mesas
2. Ajuste textos e descrições
3. Crie variações de componentes
4. Adicione interações de protótipo

### Fase 3: Desenvolvimento (2-4 horas)

1. Use **design-tokens.css** como base
2. Importe dados de **figma-landing-page-structure.json**
3. Crie componentes React/Vue baseados nas specs
4. Implemente responsividade

### Fase 4: Deploy

1. Build do projeto
2. Deploy Netlify/Vercel
3. Configurar domínio
4. SEO e analytics
